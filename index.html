What is Block 52?

Protocol & Blockchain

Block 52 is an efficient protocol to facilitate trustless player to player card games over the internet, with zero knowledge and no central authority. Block 52 is an implementation of several "mental poker" protocols with the addition of its own proof of stake blockchain.

A Single Purpose Blockchain

General purpose chains like Ethereum are not good enough.

By creating a blockchain and smart contract language with the sole purpose enabling peer to peer card games like Poker, we can create a safer and better experience.

Just like the lbry.io project is a protocol and runs its own blockchain for the distribution of digital content, block52 will allow developers and the like to build card games on top of its protocol.

Joker: A new smart contract language for card games

Joker is a simple, portable language for systematically specifying multiplayer turn-based card games using the standard 52-card deck written by researchers at Columbia University.
